# Hotel Reviews Sentiment Anlysis

## Overview

This project utilizes state-of-the-art NLP models, including BERT and RoBERTa, to perform various natural language processing tasks. The models are implemented and fine-tuned for specific downstream tasks such as sentiment analysis.

## Requirements

- Python 3.x
- PyTorch Lightning
- Transformers library
